# pipx

## Commands
```shell
pipx --help
pipx install <PACKAGE>
# Install packages into an existing Virtual Environment
pipx inject 
pipx upgrade <PACKAGE>
pipx upgrade-all
pipx reinstall
pipx reinstall-all
pipx list
# Run latest version of package in Virtual Environment
pipx run <PACKAGE>
# Run pip in an existing pipx-managed Virtual Environment
pipx runpip
```
